# 嵌入式系统推荐

## 文档定位

本文件按课程结构展开嵌入式系统方向的学习计划，适合从基础到前沿的系统化进修，并提供项目实践与突破方向。

## 课程结构

### 1. 基础阶段（第一年）
- 数字电路与模拟电路基础：逻辑门、电路分析、放大器、滤波、运算放大器。
- 单片机与MCU 原理：CPU 核心、总线结构、引脚功能、复位与中断。
- 嵌入式 C 语言基础：指针、数据结构、位操作、外设寄存器访问。
- 常用接口：GPIO、ADC、PWM、定时器、串口通信。

### 2. 专业阶段（第二年）
- 嵌入式系统开发：STM32/ESP32/NXP 芯片、开发环境配置、板级调试。
- RTOS 与实时软件：FreeRTOS 任务、队列、互斥、时间管理、优先级调度。
- 通信与网络：UART、SPI、I2C、CAN、Ethernet、Wi-Fi、BLE。
- 嵌入式硬件设计：原理图设计、PCB 布局、电源管理、EMI/EMC 基础。

### 3. 高级应用与前沿（第三年）
- 嵌入式 Linux 与 Linux 驱动：设备树、内核模块、交叉编译。
- 物联网与边缘计算：MQTT、CoAP、NB-IoT、5G、边缘推理。
- 嵌入式 AI 与 TinyML：神经网络加速、模型部署、图像识别应用。
- 安全与可靠性：Secure Boot、OTA 升级、加密通信、硬件可信根。

## 课程模块与学习内容

### 嵌入式硬件基础
- 学习内容：PCB 原理图、元器件选型、电源稳压、EMI、信号完整性规划。
- 实践：设计一个 STM32 最小系统原理图，完成基础电源和复位电路。

### MCU 与固件开发
- 学习内容：寄存器编程、外设驱动、定时器、DMA、中断服务程序。
- 实践：使用 CubeMX 或 HAL 库开发 LED、按键、ADC 采集与串口输出程序。

### RTOS 与任务调度
- 学习内容：任务创建、任务间通信、互斥锁、事件组、定时器、内存管理。
- 实践：使用 FreeRTOS 实现一个传感器采集与数据传输的多任务系统。

### 通信协议与联网
- 学习内容：串口、SPI、I2C、CAN 区别与应用；TCP/IP、MQTT、HTTP、WebSocket。
- 实践：用 ESP32 实现一个 Wi-Fi 物联网节点，上传传感器数据到云平台。

### 嵌入式 AI 与边缘推理
- 学习内容：TinyML 基础、模型压缩、边缘推理框架、加速器使用。
- 实践：基于 STM32/ESP32 实现手势识别或图像分类应用。

### 系统安全与可靠性
- 学习内容：安全启动、固件完整性、加密算法、硬件防篡改、故障恢复设计。
- 实践：实现一个带签名校验的 Bootloader 与 OTA 升级流程。

## 详细知识点展开

### MCU 体系结构与外设
- 内容：总线层次、管脚复用、存储器层次、时钟系统、调试接口。
- 应用：设计嵌入式系统时的硬件选型与性能评估。

### 无线与物联网技术
- 内容：BLE、Wi-Fi、LoRa、NB-IoT、5G 对比与典型应用。
- 应用：智能家居、工业物联网、远程监测系统。

### RISC-V 与国产生态
- 内容：RISC-V 架构基础、指令集扩展、国产芯片生态、软硬件开发链。
- 应用：国产嵌入式设备、教学实验平台、差异化技术储备。

### FPGA 与软硬件协同
- 内容：FPGA 基本原理、Verilog/VHDL、AXI 总线、硬件加速。
- 应用：高速信号处理、图像处理、专用通信协议实现。

## 项目与实践建议

### 课程项目
- 微控制器课程：完成一个带按键、显示与传感器接口的嵌入式系统。
- 嵌入式系统课程：设计一个通信节点，支持 UART、SPI、I2C 数据传输。
- 物联网课程：实现一个远程数据采集与云端展示应用。

### 实训与竞赛项目
- 智能传感器网关：采集环境参数并通过 MQTT 上传云端。
- 嵌入式 AI 设备：手势识别、语音唤醒或异常检测系统。
- 车载电子子系统：仪表盘、驾驶辅助模块、车载通信网关。
- 安全固件项目：Secure Boot、OTA 升级、固件签名验证。
