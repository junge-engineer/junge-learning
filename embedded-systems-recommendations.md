# 嵌入式系统推荐

## 核心知识点

- 数字电路与模拟电路基础
- 单片机原理与接口技术
- 嵌入式操作系统与实时操作系统（RTOS）
- 嵌入式硬件设计与PCB基础
- 传感器、驱动电路与信号采集
- ARM/STM32/ARM Cortex-M处理器架构
- 嵌入式软件开发、调试与优化
- 通信接口：UART、SPI、I2C、CAN、Ethernet、USB
- 嵌入式系统安全与加密
- 低功耗设计与电源管理
- 无线通信：BLE、Wi-Fi、LoRa、NB-IoT、5G

## 细分知识点

- MCU体系结构、总线与中断机制
- 引导加载程序（Bootloader）与固件升级
- RTOS任务调度、线程同步、互斥与消息队列
- 嵌入式Linux与开发板系统集成
- 外设接口：ADC、DAC、PWM、定时器、DMA
- 传感器融合与滤波算法
- 机器视觉与图像处理基础
- 嵌入式人工智能：TinyML、边缘推理、神经网络加速
- FPGA/SoC与软硬件协同设计
- 物联网协议：MQTT、CoAP、Modbus、OPC UA
- 自动化测试与持续集成

## 重要技能

- C/C++语言编程与嵌入式开发
- Python脚本与自动化工具使用
- STM32、ESP32、NXP、TI等平台开发实操
- Keil MDK、IAR EWARM、CubeMX、PlatformIO、VS Code
- 嵌入式调试、JTAG/SWD、逻辑分析仪应用
- PCB设计与电源完整性分析
- 实时系统响应与中断调度设计
- 传感器驱动与数据采集系统实现
- 无线模块与网络通信开发
- 低功耗及节能设计方法
- 系统安全：安全启动、加密、固件完整性

## 前沿方向与“卡脖子”技术

- RISC-V芯片与国产MCU生态
- 嵌入式AI芯片与边缘计算加速器
- 高可靠性工业级处理器与安全芯片
- 核心传感器芯片：MEMS、惯性测量单元、图像传感器
- 高速ADC/DAC、射频芯片与高精度采集模块
- 车规级MCU与车载以太网
- 物联网安全、可信计算与硬件根信任
- 5G/6G通信模组与低时延边缘节点
- FPGA+软件异构架构与硬件加速
- 智能终端固件 OTA、远程升级与安全管理

## 推荐项目

- STM32/ESP32智能传感器网关
- 嵌入式AI边缘设备：人脸识别、手势识别、异常检测
- 车载仪表或智能驾驶辅助子系统
- 物联网数据采集与远程监控系统
- 无线智能家居设备与低功耗传感网络
- FPGA信号处理与高速数据采集原型
- 安全固件：加密通信、Secure Boot与防篡改设计
- 智能可穿戴设备或健康监测模块
- 机器人运动控制与多传感器融合
