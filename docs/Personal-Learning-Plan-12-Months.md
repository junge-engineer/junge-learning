# 个人学习计划：12 个月周计划（机电一体化 + 数控 + 嵌入式 + 电气自动化）

## 目标
- 通过 12 个月系统学习，形成“数控加工 + 嵌入式控制 + PLC 电气自动化 + IIoT 采集”个人能力。
- 以项目为驱动，每个月完成一个小实践，最后交付一个完整的期末案例样机。
- 适合个人自学：坚持每周 10–15 小时，结合硬件实操与文档输出。

## 学习节奏
- 周一至周五：每天 1–1.5 小时（理论+练习/阅读）。
- 周末：每个周末 3–4 小时（实验/动手/项目实施）。
- 建议工具：GitHub 或本地 Git，写周报；使用笔记记录思路和问题。

## 总体能力结构
- 基础能力：C 语言、电子电路、机械制图、PLC 梯形图。
- 动手能力：STM32 外设、PWM/编码器、PLC 控制、数控 CAM、伺服调参。
- 集成能力：PLC + 嵌入式 + HMI + 工业网络。
- 拓展能力：视觉检测、数据采集、维护与优化。

## 月度主题与每周任务

### 第 1 月：打好底座

#### 周 1：C 语言与编程工具
- 学习内容：变量、流程语句、函数、数组、指针初步。
- 练习：完成两个简单程序：计算器和字符串处理。
- 验收：能读懂他人 C 代码并修改一个例程。

#### 周 2：嵌入式开发环境搭建
- 学习内容：STM32CubeIDE / PlatformIO 环境安装，开发板连接。
- 练习：加载 Blink 和 UART 例程。
- 验收：开发板能跑起灯闪与串口输出日志。

#### 周 3：电子基础与测量
- 学习内容：电压、电流、电阻、欧姆定律、常用元件识别。
- 练习：面包板上完成分压电路与简单 RC 滤波电路。
- 验收：使用万用表和示波器测量，并记录实验数据。

#### 周 4：版本控制与项目记录
- 学习内容：Git 基本命令、仓库结构、README 写法。
- 练习：将第一个实验写入 GitHub repo，撰写周报。
- 验收：完成一个 GitHub 项目并提交周报文档。

### 第 2 月：STM32 外设与基础控制

#### 周 5：GPIO 与中断
- 学习内容：输入/输出、上拉/下拉、中断原理。
- 练习：实现按键控制 LED 状态机。
- 验收：按键可切换三种 LED 模式并稳定工作。

#### 周 6：PWM 与电机控制
- 学习内容：PWM 原理、占空比、频率、驱动电机注意事项。
- 练习：用 PWM 驱动直流电机/步进电机，并观察转速变化。
- 验收：实现 PWM 控制转速并记录占空比与转速关系。

#### 周 7：ADC 与传感器
- 学习内容：ADC 采样、传感器接口、数据滤波。
- 练习：读取光敏/电位器/温度传感器，并显示数值。
- 验收：实现一个可视化数据输出的仪表盘（串口/屏幕）。

#### 周 8：编码器与定时器
- 学习内容：编码器工作原理、定时器中断、计时与测速。
- 练习：实现编码器测速模块。
- 验收：输出稳定的转速值并计算测量误差。

### 第 3 月：闭环控制与通信

#### 周 9：PID 控制基础
- 学习内容：PID 原理、P/I/D 各项作用、调参方法。
- 练习：写出一个 PID 调参步骤与实验计划。
- 验收：理解三种参数对响应的影响。

#### 周 10：STM32 上实现 PID
- 学习内容：控制回路设计、信号滤波、采样周期。
- 练习：用 PID 控制电机速度或位置。
- 验收：实现稳态误差小于 5% 的速度控制。

#### 周 11：串口通信与简单协议
- 学习内容：UART/RS232、Modbus RTU、串口调试工具。
- 练习：实现 STM32 与 PC 之间的通信命令。
- 验收：从 PC 发送命令并正确触发 STM32 反应。

#### 周 12：闭环控制小项目
- 项目：STM32 + 编码器 + PID 位置控制。
- 练习：实现目标位置设定与到位判定。
- 验收：目标位置准确到 ±2cm（或自定义精度），并撰写项目报告。

### 第 4 月：PLC 与电气自动化基础

#### 周 13：PLC 梯形图基础
- 学习内容：继电器逻辑、定时器、计数器。
- 练习：在 PLC 仿真器中实现启停逻辑。
- 验收：能熟练绘制三段常用逻辑图。

#### 周 14：低压电气与安全
- 学习内容：接触器、继电器、熔断器、电气图识读。
- 练习：拆解一张电气图并标注各元件功能。
- 验收：能写出电气保护与互锁方案。

#### 周 15：PLC 传感器与执行器
- 学习内容：输入输出模块、传感器接线、埋线注意事项。
- 练习：搭建传感器触发的示例电路。
- 验收：实现一个传感器驱动电机的完整流程。

#### 周 16：PLC 小项目
- 项目：输送带启停与分拣逻辑。
- 练习：写逻辑、仿真、测试。
- 验收：实现“有料进站→输送→分拣输出”的功能。

### 第 5 月：数控与 CAM 入门

#### 周 17：机械制图与工件基准
- 学习内容：二维图纸、几何尺寸与公差、装夹基准。
- 练习：完成一个简单零件的二维图草案。
- 验收：定位基准正确，尺寸标注完整。

#### 周 18：CAM 刀路生成
- 学习内容：CAM 工作流、刀具路径类型、切削参数。
- 练习：用 Fusion 360/Mastercam 生成二维刀路。
- 验收：生成的 G-code 无碰撞，仿真正常。

#### 周 19：G-code 理解
- 学习内容：G0/G1/G2/G3、M 代码、进给、主轴控制。
- 练习：手写一段简单二维轮廓刀路。
- 验收：仿真器正确解析并执行。

#### 周 20：数控加工演示
- 项目：加工一个简单夹具或工件。
- 练习：生成刀路、调机床、记录加工参数。
- 验收：工件达到预期尺寸或误差可控制。

### 第 6 月：伺服与运动控制

#### 周 21：伺服驱动理论
- 学习内容：伺服系统构成、编码器反馈、增益与回路。
- 练习：对比伺服与步进控制的优缺点。
- 验收：能选择合适伺服参数用于样机。

#### 周 22：伺服调参实训
- 学习内容：比例、积分、微分调参；响应速度与稳定性。
- 练习：调试伺服驱动器的 P/I/D 参数。
- 验收：系统能稳态运行且无明显振荡。

#### 周 23：轨迹与插补
- 学习内容：速度规划、S-曲线、限速处理。
- 练习：实现两段连续运动并观察过渡。
- 验收：运动平稳，无冲击。

#### 周 24：伺服整合实验
- 项目：STM32 发送目标位置给伺服驱动并闭环执行。
- 练习：实现伺服对位并记录误差。
- 验收：达到预期位置精度并写调试记录。

### 第 7 月：工业网络与 IIoT

#### 周 25：Modbus 与现场总线
- 学习内容：Modbus RTU/TCP、地址映射、通信周期。
- 练习：实现 STM32 与 PC 之间 Modbus 通信。
- 验收：可读取/写入寄存器数据。

#### 周 26：MQTT 与边缘采集
- 学习内容：MQTT 原理、Broker、Topic 结构。
- 练习：在 Raspberry Pi / 树莓派上运行 Mosquitto。
- 验收：设备数据成功发布到 Broker。

#### 周 27：Node-RED 与可视化
- 学习内容：Node-RED 流程、数据处理、Dashboard。
- 练习：把 MQTT 数据显示成趋势图。
- 验收：实现一个可视化的实时仪表盘。

#### 周 28：IIoT 小项目
- 项目：生产数据采集与状态监控。
- 练习：编码器/传感器数据上报并可视化。
- 验收：Dashboard 能显示当前状态和累计统计。

### 第 8 月：HMI/SCADA 与系统集成

#### 周 29：HMI 设计与交互
- 学习内容：画面布局、报警显示、控制按钮。
- 练习：设计 HMI 画面原型。
- 验收：画面清晰并能表达工艺状态。

#### 周 30：PLC 与 HMI 联调
- 学习内容：标签映射、数据交换、画面同步。
- 练习：实现启动/停止按钮与状态反馈。
- 验收：HMI 按钮可控制 PLC，状态同步正常。

#### 周 31：SCADA 监控基础
- 学习内容：远程监控架构、数据采集策略。
- 练习：使用轻量 SCADA 软件监控一个节点。
- 验收：可远程查看设备数据并记录报警。

#### 周 32：集成方案设计
- 学习内容：系统架构、通信流程、接口定义。
- 练习：写一份该样机的系统架构文档。
- 验收：架构文档完成并可做为实现参考。

### 第 9 月：机器视觉入门

#### 周 33：图像处理基础
- 学习内容：灰度、阈值、边缘检测、轮廓识别。
- 练习：用 OpenCV 实现工件轮廓识别。
- 验收：正确检测到至少一种工件形态。

#### 周 34：特征匹配与定位
- 学习内容：模板匹配、ORB、角点检测。
- 练习：实现工件定位并输出坐标。
- 验收：定位误差小于 5mm（或符合硬件精度）。

#### 周 35：相机标定与坐标转换
- 学习内容：相机内参、畸变校正、像素映射。
- 练习：完成一个简单标定并计算转换矩阵。
- 验收：能把像素坐标转换为实际毫米坐标。

#### 周 36：视觉+控制集成
- 项目：视觉检测后触发 PLC/STM32 动作。
- 练习：实现“检测结果 → 对位/报警”闭环。
- 验收：视觉结果可控制后端执行器。

### 第 10 月：诊断与优化

#### 周 37：故障诊断流程
- 学习内容：故障分类、日志记录、复现步骤。
- 练习：设计一个故障诊断表格。
- 验收：能完整写出一次故障诊断过程。

#### 周 38：数据分析基础

- 练习：对采集到的数据做统计分析。
- 验收：输出异常报警报告。

#### 周 39：预测维护规则
- 学习内容：阈值模型、简单规则引擎、报警策略。
- 练习：实现温度/振动阈值报警脚本。
- 验收：系统可自动生成维护提醒。

#### 周 40：性能优化实验
- 学习内容：加工参数优化、PID 参数对比、响应时间改善。
- 练习：优化一个控制环路或加工工艺。
- 验收：记录优化前后性能提升。

#### 周 41：期末案例规划

- 验收：项目计划书包含功能列表、接口、时间表。

#### 周 42：硬件搭建与接线
- 学习内容：框架搭建、电气布线、传感器安装。
- 练习：搭建送料机制与传感器结构。
- 验收：完成物料清单并拍照记录。

#### 周 43：控制软件开发
- 学习内容：PLC 程序、嵌入式固件、HMI 脚本。
- 练习：完成主控制程序与通信协议。
- 验收：系统能本地联调。

## 期末案例：迷你智能装配线样机

### 目标
- 设计一个具备“自动送料 → 视觉检测 → 伺服对位 → 下料统计”功能的样机。
- 输出完整技术文档、代码仓库和演示视频。

### 功能点
- 进料识别与控制输送带
- 伺服位置对位与闭环反馈
- 视觉检测结果判断 OK/NG
- 结果统计与 IIoT 数据上报
- HMI 状态显示与启动/停止控制

### 评估标准（100 分）
- 功能完整性：30
- 稳定性与精度：25
- 文档与代码质量：15
- 创新与优化：15
- 演示表达：15

### 关键里程碑
1. 系统设计与需求确认
2. 硬件搭建与接线
3. 控制与通信程序开发
4. 视觉检测与 HMI 集成
5. 测试、优化与演示
