# MAE104 自动控制原理 / Control Theory

## 课程目标 / Course Objectives
- 掌握控制系统建模与时域/频域分析方法
- 能设计与调试 PID 控制器并理解现代控制基础
- 能使用仿真工具验证控制方案

## 每周主题（摘要） / Weekly topics (summary)
1. 控制系统概述与建模（传递函数法）
2. 时域分析：单位响应、稳态误差
3. 根轨迹法与系统稳定性判定
4. 频域分析：幅频、相频、奈奎斯特判据
5. PID 控制器原理与参数整定方法
6. 复合控制与补偿网络
7. 状态空间基本概念、可控性、可观性
8. 离散时间系统与数字控制简介
9. 非线性系统概述与线性化方法
10. 伺服系统与位置控制实例
11. 嵌入式控制器实现注意事项
12. 控制系统仿真实践（Simulink）
13. 课程项目：设计与实现一个闭环控制系统
14. 复习与期末考试

## 实验与作业 / Labs & Assignments
- Simulink 建模作业、PID 调参实验、控制器实现报告
