# MAE201 传动与动力学（进阶）

## 课程目标 / Course Objectives
- 深入理解动力学建模与振动分析方法
- 能从系统角度设计高可靠传动方案并进行仿真验证

## 每周主题（摘要） / Weekly topics (summary)
1. 动力学建模方法与自由度分析
2. 转动惯量与等效质量
3. 振动系统建模：单自由度、多自由度
4. 阻尼、谐振与频率响应
5. 转矩脉动与扭振分析
6. 动平衡与不平衡力分析
7. 传动齿轮动力学与接触疲劳
8. 高速机械的设计要点
9. 精密定位与误差模型
10. 伺服系统动态特性与带宽分析
11. 兼容性设计与系统集成考虑
12. FEA 在动力学中的应用案例
13. 课程项目：动力学仿真与实验对比
14. 复习与期末答辩

## 实验与作业 / Labs & Assignments
- 转动惯量测量、振动台测试、仿真报告
