# 机电专业课程概览 / Program Overview

本章节为机械与自动化专业的课程总览，适合作为学生或教师快速查阅课程设置与学习路径。

This page summarizes the program structure and suggested learning progression.

## 学年安排（建议） / Suggested progression

- Year 1: Foundation — mathematics, physics, mechanical drawing, basic materials and manufacturing.
- Year 2: Core — mechanical design, transmissions, control theory, sensors, PLC & automation.
- Year 3: Advanced — robotics, intelligent manufacturing, digital twin, advanced manufacturing.

## 如何使用本资料 / How to use

1. 选一门课程作为当期重点（例如 MAE101）。
2. 按周次安排完成课内理论、实验与小项目。
3. 形成项目档案（报告、BOM、代码、演示视频），用于实习/毕业设计/竞赛申请。
