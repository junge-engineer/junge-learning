# MAE102 工程力学与材料力学

## 课程目标 / Course Objectives
- 掌握静力学与材料力学的基本理论
- 能进行常见零件的强度校核与疲劳分析
- 能使用有限元工具进行简单结构分析

## 每周主题（摘要） / Weekly topics (summary)
1. 力、力矩与静力平衡
2. 支反力、自由体图、梁内力
3. 剪力-弯矩图、梁的弯曲应力
4. 扭转与轴的强度计算
5. 应力应变曲线、弹性与塑性
6. 强度理论（最大主应力、最大剪应力）
7. 挠度计算、变形能方法
8. 疲劳基础、疲劳寿命估算方法
9. 连接件分析：焊缝、螺栓、键联
10. 热应力与多场耦合概念
11. 复合材料力学概述（选讲）
12. 有限元建模入门（网格/边界条件基础）
13. 课程实验：材料疲劳试验/应变片测量
14. 复习与期末考核

## 实验与作业 / Labs & Assignments
- 梁弯曲实验、应变计测量、材料力学试验
- FEA 简单作业与数据分析报告
