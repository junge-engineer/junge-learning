# MAE103 机械原理与传动系统

## 课程目标 / Course Objectives
- 掌握常见机械机构的运动学与动力学分析方法
- 能进行齿轮、轴系与传动装置的设计计算
- 能分析传动系统的效率与振动问题

## 每周主题（摘要） / Weekly topics (summary)
1. 机构基础与等效约束
2. 连杆机构速度与加速度分析
3. 凸轮机构与运动合成
4. 齿轮基本参数与几何学
5. 齿轮设计与强度计算
6. 蜗轮蜗杆、同步带/链传动
7. 轴承与润滑基础
8. 传动误差与补偿技术
9. 动力传递与效率计算
10. 机械振动基础与动平衡
11. 精密传动系统案例分析
12. 传动箱/减速器设计综合练习
13. 课程项目：减速器设计报告提交
14. 复习与期末答辩

## 实验与作业 / Labs & Assignments
- 传动部件计算题、减速器设计项目
- 振动测量实验与数据分析
