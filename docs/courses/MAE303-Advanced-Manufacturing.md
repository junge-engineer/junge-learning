# MAE303 高级制造技术

## 课程目标 / Course Objectives
- 学习先进制造技术，包括增材制造、微细加工与高精度加工工艺
- 掌握制造过程中的质量控制与误差源分析方法

## 每周主题（摘要） / Weekly topics (summary)
1. 高级制造概览：增材制造、精密加工
2. 3D 打印工艺比较与材料选择
3. 微细加工与 MEMS 制造简介
4. 加工中心与高精度刀具技术
5. 表面处理、热处理对材料性能的影响
6. 工艺参数对加工精度的影响分析
7. 多轴加工与夹具设计要点
8. 检测与质量控制技术
9. 复合制造与复合材料加工
10. 自动化检测与在线质量控制
11. 新材料与加工挑战（碳纤维、陶瓷）
12. 项目：增材制造工艺开发或工艺优化
13. 项目实施与测试
14. 复习与成果展示

## 实验与作业 / Labs & Assignments
- 3D 打印工艺参数实验、加工精度测试
