# MAE302 智能制造与数字孪生

## 课程目标 / Course Objectives
- 理解工业 4.0 架构、数字孪生技术与生产线优化方法
- 能搭建小型数字孪生原型并实现数据映射与虚拟调试

## 每周主题（摘要） / Weekly topics (summary)
1. 智能制造概述与工业 4.0 基本概念
2. 生产线自动化与柔性制造体系
3. 传感器网络与实时数据采集
4. 工业通信协议：OPC UA、EtherCAT、Profinet
5. MES/SCADA 系统简介与功能
6. 数字孪生概念、建模方法与数据映射
7. 虚拟调试与实体同步技术
8. 生产优化：瓶颈识别与调度策略
9. 预测维护与状态监测
10. 工业大数据与简单分析方法
11. 安全与权限管理、网络隔离
12. 项目：搭建小型数字孪生原型（仿真+数据流）
13. 项目调试与结果分析
14. 总结、未来趋势讨论

## 实验与作业 / Labs & Assignments
- 数字孪生原型搭建、数据可视化实验
