# MAE301 机器人学（上）——运动学与轨迹规划

## 课程目标 / Course Objectives
- 掌握机器人运动学、坐标变换与 DH 参数法
- 能解正/反运动学问题并实现基本轨迹规划与插补算法
- 熟悉视觉伺服的基本原理并能在仿真环境中验证算法

## 每周主题（摘要） / Weekly topics (summary)
1. 机器人简介与常见构型
2. 坐标系与齐次变换
3. 正运动学建模与 DH 参数法
4. 反运动学方法与数值解法
5. 运动学奇异性与解的存在性分析
6. 速度学：雅可比矩阵与速度/力学关系
7. 运动学仿真工具与验证
8. 轨迹插补：直线、圆弧、样条
9. 轨迹规划与路径优化
10. 运动学在并联机器人中的应用
11. 视觉伺服与基于视觉的误差校正
12. 实验：机器人轨迹仿真与离线编程
13. 项目：机械臂抓取任务实现
14. 复习与课程考核

## 实验与作业 / Labs & Assignments
- ROS 仿真实验、轨迹插补代码、抓取任务实现报告
