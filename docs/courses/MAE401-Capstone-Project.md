# MAE401 毕业设计 / Capstone Project

## 课程目标 / Course Objectives
- 综合运用所学知识完成一个从需求到实现的机电系统
- 掌握项目管理、技术文档编写与团队协作技能

## 每周安排（摘要） / Weekly topics (summary)
1. 选题、需求分析与系统分解
2. 技术方案设计与可行性评估
3. 建模与设计细化
4. 原型制作计划与物料采购（BOM）
5. 原型制作与单元测试
6. 控制系统集成与通信调试
7. 结构装配与性能软硬件验证
8. 数据采集与系统调优
9. 性能测试与可靠性评估
10. 文档编写：用户手册、维护手册
11. 项目演示准备与PPT制作
12. 预答辩与评审反馈
13. 正式答辩与演示
14. 项目结题与成果归档

## 成果要求 / Deliverables
- 技术报告（论文）、演示视频、源码与PCB文件、BOM、测试报告
