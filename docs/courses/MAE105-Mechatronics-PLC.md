# MAE105 机电一体化与 PLC 实验 / Mechatronics & PLC

## 课程目标 / Course Objectives
- 理解机电一体化系统的组成与基本工作原理
- 掌握 PLC 梯形图编程并能搭建小型工业控制系统
- 熟悉 HMI 与现场总线的基本使用

## 每周主题（摘要） / Weekly topics (summary)
1. 机电一体化系统概述
2. PLC 基本结构与工作原理
3. 梯形图编程基础：逻辑与触点
4. 计时器、计数器与数据处理
5. I/O 扩展与通信接口
6. HMI 概述与简单界面设计
7. 现场总线与工业以太网简介
8. 实验：油压/气压控制回路整合
9. 实验：传送带启动/停机/故障处理程序
10. 实验：位置控制与简单伺服调试
11. 实验：报警与安全互锁设计
12. 项目：小型自动输送线搭建
13. 项目调试与演示
14. 总结与评估

## 实验与作业 / Labs & Assignments
- 每周上机实验、期末小组项目（输送线）
