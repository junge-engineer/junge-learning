# 机电一体化与智能制造课程整合方案

概述
- 目标：在现有课程基础上，整合数控技术、嵌入式系统与电气自动化，形成模块化、项目驱动的教学路径，培养复合型岗位人才。
- 说明：下列建议已尽量映射到现有课程（链接到校内课程文件），并补充需要新增的实践模块与项目。

一、能力目标（毕业生应掌握）
- 数控加工与编程、刀具与工艺参数选择
- 嵌入式控制（ARM/STM32）、外设驱动与运动控制
- 电气控制与PLC编程、低压电气设计与调试
- 系统集成（PLC+伺服+HMI）、工业网络与数据采集（IIoT）
- 工程实践能力、故障诊断与团队项目交付

二、与现有课程的映射（建议主干课程与说明）
- [MAE101-Mechanical-Drawing.md](courses/MAE101-Mechanical-Drawing.md)：保留作图与装配基础，加入数控加工装配图的标注练习。
- [MAE102-Engineering-Mechanics.md](courses/MAE102-Engineering-Mechanics.md)：作为力学基础，支持运动学与结构强度课程中的机械设计案例。
- [MAE103-Mechanism-Transmission.md](courses/MAE103-Mechanism-Transmission.md)：结合传动系统选择与伺服驱动匹配的实例教学。
- [MAE104-Control-Theory.md](courses/MAE104-Control-Theory.md) 及 [MAE104-TeachPlan.md](docs/courses/MAE104-TeachPlan.md)：扩展为控制理论+运动控制实训，加入PID与伺服调参实验。
- [MAE105-Mechatronics-PLC.md](courses/MAE105-Mechatronics-PLC.md)：作为机电一体化核心，增加PLC与嵌入式交互模块（串口/Modbus通信）。
- [MAE201-Transmission-Dynamics.md](courses/MAE201-Transmission-Dynamics.md)：用于高年级机械动态与振动控制的案例分析。
- [MAE301-Robotics-Motion.md](courses/MAE301-Robotics-Motion.md)：引入机器人示教、轨迹规划与视觉定位接口实践。
- [MAE302-Intelligent-Manufacturing.md](courses/MAE302-Intelligent-Manufacturing.md)：作为智能制造专题，覆盖IIoT、MES对接与数字化生产线设计。
- [MAE303-Advanced-Manufacturing.md](courses/MAE303-Advanced-Manufacturing.md)：高级加工工艺、刀具寿命管理与工艺优化。
- [MAE401-Capstone-Project.md](courses/MAE401-Capstone-Project.md)：毕业设计改为“产线集成+智能化改造”方向项目。

三、建议新增模块与课程单元（可作为选修或集成周）
- 嵌入式控制实训（建议学时：40）——内容：STM32基础、外设驱动、PWM与编码器、串口/以太网通信、RTOS简介；成果：步进/伺服位置控制实验。
- 数控CAM与刀具工艺（学时：36）——内容：CAD/CAM流程、刀路生成、加工仿真、装夹与工件检验；成果：学生完成指定零件加工并提交检验报告。
- 伺服与运动控制（学时：30）——内容：伺服选型、驱动调参、闭环控制、运动曲线规划；成果：伺服调参与误差分析实验。
- 工业网络与IIoT（学时：24）——内容：Modbus/Profinet/OPC-UA、MQTT上报、边缘网关示例；成果：小型数据采集与可视化演示。
- 机电一体化集成实训（学时：60）——内容：电气柜设计、PLC+HMI联调、机械与嵌入式联动；成果：流水线小样机集成项目。

四、阶段化学习路线（示例：2 年制/4 学期）
- 第1学期（基础）：机械制图([MAE101])、工程力学([MAE102])、电工基础（新增实验）、C语言基础。
- 第2学期（技能入门）：数控基础、电子与嵌入式导论（新增）、PLC入门([MAE105])、CAM入门。
- 第3学期（技能提升）：嵌入式控制实训、伺服与运动控制、数控加工实训、工业网络入门。
- 第4学期（集成与实习）：机电一体化集成实训、毕业项目([MAE401])、企业实训/顶岗实习、证书考取准备。

五、示范项目（可直接作为课程考核）
- 项目A（初级）：基于STM32的步进电机精确定位装置（目标：实现位置控制与基本人机交互）。
- 项目B（中级）：小型数控加工线——CAD出图→CAM编程→数控加工→装配测量（目标：零件合格率≥90%）。
- 项目C（高级）：PLC+伺服+HMI的自动送料与装夹单元，并通过边缘网关上报生产数据（目标：实现远程监控与基础故障报警）。

六、考核与企业对接建议
- 考核方式：理论考试（30%）+实训考核（50%）+项目报告与答辩（20%）。
- 证书方向：建议与西门子/三菱/艾默生等厂商合作提供厂商培训与认证；推广数控操作证、低压电工证、嵌入式开发证书。
- 校企合作：建议引入企业真实设备作为训练场、建立订单班、开展企业导师联合指导。

七、教学资源与设备清单（优先级）
- 必备：数控车/铣机床（1台实训线或多台台架）、PLC训练箱、伺服驱动试验台、STM32开发板与示波器. 
- 推荐：数控仿真软件（Mastercam/CAM软件授权）、边缘计算网关、工业相机用于视觉教学。
