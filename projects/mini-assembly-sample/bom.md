# BOM — 低成本个人套件建议

- STM32 开发板（如 STM32F103C8T6 BluePill 或 Nucleo-F401RE） x1
- 编码器套件（增量编码器 + 配套轴联接件） x1
- 步进/伺服电机套件（含驱动模块） x1
- 小型直流电机（练习 PWM） x1
- 电源模块（5V/12V 可调） x1
- PLC 模拟训练板或小型 PLC（西门子 LOGO!/入门 PLC） x1
- 树莓派 4 / 树莓派 Zero（做边缘网关） x1
- USB 示波器或台式示波器（入门级） x1
- 杜邦线、面包板、传感器（限位开关、光电传感器）
- 工具：电烙铁、万用表、螺丝刀套装

采购建议
- 国内平台（淘宝/京东/阿里巴巴/拉钩）常有套件，优先选带评价与技术支持的店铺。
- 伺服与数控相关的驱动建议从正规渠道采购或学校借用真实机床设备以保证安全。
